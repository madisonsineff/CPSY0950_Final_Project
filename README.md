# CPSY0950_Final_Project -- GamePython Hub
Madison Sineff and Pearl Chia  

## Project Overview
Our GamePython app is a collection of interactive mini-games inspired by the online messgae GamePigeon. The project includes a main hub screen where users can choose from multiple games, including Connect Four, Anagrams, Filler, and Cup Pong. The purpose of this project was to practice Python programming, interactive game design, Pygame, project organization, and debugging across multiple files. We wanted to incorporate aspects of vibecoding and learn to apply technical frameworks/POVs to it.

## Requirements
To run this project, you need:
- Python 3
- Pygame or Pygame-ce

No additional datasets or data files are required for this project. All images and audio files needed to run the games are included in the project folder.

### Main Hub File Structure

- `GamePython_MAINHUB.py`  
  Main file for the project. This file opens the Game Pigeon Hub interface and allows the user to select which mini-game to play.

- `hub_music.mp3`  
  Audio file used for the main hub.

### Connect Four Files

- `connectfour.py`  
  Python file containing the Connect Four game. This game uses a grid-based board where players take turns placing pieces and trying to get four in a row.

- `connect4_audio.mp3`  
  Audio file used for the Connect Four game.

- `connect4background.jpg`  
  Background image for the Connect Four game -- not used.

### Anagrams / Word Hunt Files

- `anagrams.py`  
  Python file containing the Anagrams/Word Hunt game. This game focuses on word-based user interaction.

- `Anagrams_audio.mp3`  
  Audio file used for the Anagrams/Word Hunt game.

- `wordhuntbackground.png`  
  Background image for the Anagrams/Word Hunt game -- not used.

### Filler Files

- `filler.py`  
  Python file containing the Filler game. This game involves selecting colors and updating the game board based on the player’s choices.

- `filler_audio.mp3`  
  Audio file used for the Filler game.

- `filler_background.jpg`  
  Background image for the Filler game -- not used.

### Cup Pong Files

- `cuppong.py`  
  Python file containing the Cup Pong game. This game uses visual interaction and user input to simulate a cup pong-style game.

- `cup_pong_audio.mp3`  
  Audio file used for the Cup Pong game.

- `cup_pong_background.webp`  
  Background image for the Cup Pong game -- not used.

- `cuppongbackground.webp`  
  Additional background image for the Cup Pong game -- used!

## How to Play

Start the project by running:

```bash
python GamePython_MAINHUB.py
```

From the main hub, choose the game you want to play.

General controls depend on the game, but may include:

- Mouse clicks to choose buttons, colors, pieces, or targets
- Keyboard input for word-based games
- On-screen game instructions and feedback

## Notes for Running

All Python files, images, and audio files should stay in the same project folder unless the code is updated to use different file paths.

If an image or audio file is missing, the game may not load correctly. Make sure the full repository is downloaded before running the program.

If the program does not run, check that Python and Pygame are installed correctly and that the command is being run from inside the project folder.
